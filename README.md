# ⚡ Калькулятор мощности ИБП

Простое и лёгкое веб-приложение для расчёта необходимой мощности источника бесперебойного питания (ИБП/UPS) под ваш набор оборудования. Работает полностью в браузере, не требует установки, сервера и интернет-соединения.

## 🚀 Возможности

- 📦 **Библиотека устройств**: готовые профили для ПК, мониторов, принтеров, сетевого оборудования, серверов и др.
- ✏️ **Свои значения**: возможность добавить любое устройство с указанием собственной мощности
- ⚖️ **Гибкие настройки**: настраиваемый запас мощности (%) и коэффициент мощности (Power Factor)
- 🔄 **Мгновенный расчёт**: результаты обновляются в реальном времени при любом изменении
- 📱 **Адаптивный дизайн**: корректно отображается на десктопах, планшетах и смартфонах
- 🔌 **Без зависимостей**: чистый HTML/CSS/JS, работает оффлайн из локальной папки
